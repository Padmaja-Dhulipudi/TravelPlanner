# Andhra Pradesh Travel Planner

A web-based travel planning application built using Python and Streamlit. The application helps users plan trips across major destinations in Andhra Pradesh, starting from Kakinada, by providing transport options, hotel recommendations, tourist attractions, real-time weather information, and a basic travel itinerary.

## Live Demo
https://travelplannerforandhrapradesh.streamlit.app/

## Technologies Used
- Python
- Streamlit
- Requests
- OpenWeatherMap API

## Features
- Destination selection across major cities in Andhra Pradesh
- Displays all available transportation options from the starting location
- Hotel listings with ratings and price per night
- City-wise tourist attractions
- Real-time weather information
- Automatically generated 3-day itinerary
- Clean and responsive user interface

## Supported Destinations
- Kakinada
- Annavaram
- Lambasinghi
- Visakhapatnam
- Rajahmundry
- Araku

## Installation
git clone https://github.com/Padmaja-Dhulipudi/TravelPlanner.git  
cd TravelPlanner  
pip install -r requirements.txt  
streamlit run app.py  

## Configuration
OPENWEATHER_API_KEY = "your_api_key_here"
